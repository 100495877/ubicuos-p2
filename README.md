----------------------------------MANUAL DE EJECUCIÓN----------------------------------

REQUISITOS:
-. Google DevTools
-. Telefono android con Depuración USB y Opciones de Desarrollador activadas
-. NodeJS

PASOS:
-. Abrir chrome://inspect y habilitar port-forwarding
-. Conectar el dispositivo móvil al PC y asegurar que aparece en chrome://inspect 
-. Ejecutar archivo index.js en el modulo server con el comando node (node server/index.js)
-. Abrir puerto display en Chrome en el PC
-. Abrir puerto local abierto tras el port-forwarding en Chrome en el dispositivo móvil
