# Decisiones técnicas

## Uso de una única aplicación web
Permite simplificar el prototipo y centrarse en la experiencia.

## Uso de Socket.IO
Facilita la sincronización en tiempo real.

## Uso de datos simulados
Permite demostrar el sistema sin depender de fuentes externas.

## Uso del móvil como controlador
Simula la cartera sin necesidad de hardware adicional.

## Uso del portátil como pantalla
Permite representar la interfaz ambiental.
