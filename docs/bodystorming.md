# Bodystorming

## Objetivo
Explorar cómo interactuaría el usuario físicamente con el sistema.

## Pruebas realizadas
- Simulación de uso del móvil como cartera
- Movimientos de inclinación para navegación
- Gestos para selección

## Observaciones
- Movimientos complejos no eran fiables
- Gestos simples funcionan mejor
- El feedback inmediato es clave

## Decisiones
- Se simplificaron los gestos
- Se priorizó claridad sobre complejidad
